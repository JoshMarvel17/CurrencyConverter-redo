This is Assignment 4 for CPSC 411.

The purpose of this application is a currency converter using the yahoo query language.

The app will first tell the user to swipe left to select the currencies you want to use.  Next the user swipes right to go back to the main screen.
After that the user will select the button to add the currencies to the pickers.

After choosing a home and foreign currency.  User Click convert and is displayed the converted currency
